# Getting Started with Axebase

This guide will help you get started with Axebase quickly and easily.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18.x or higher
- npm or yarn package manager
- Basic knowledge of React and Next.js

## Installation

### Clone the Repository

Since Axebase is a private project, you'll need to clone it from GitHub:

```bash
git clone https://github.com/Pranjal6955/axebase.git
cd axebase
```

### Install Dependencies

Choose your preferred package manager:

**Using npm:**
```bash
npm install
```

**Using yarn:**
```bash
yarn install
```

**Using pnpm:**
```bash
pnpm install
```

## Setup Your Project

Follow these steps to set up your Axebase project:

1. **Environment Configuration**

```bash
cp .env.example .env
```

Edit the `.env` file and add your Database URL, AI API keys, and other required credentials.

2. **Prepare Database**

```bash
npx prisma db push
```

3. **Start the development server**

```bash
npm run dev
```

4. **Open your browser**

Navigate to `http://localhost:3000` to see your app running!

## Project Structure

Your Axebase project will have the following structure:

```
my-app/
├── src/
│   ├── app/
│   │   ├── layout.tsx
│   │   └── page.tsx
│   └── components/
├── public/
├── package.json
└── next.config.js
```

## Configuration

Create a configuration file `axe.config.js` in your project root:

```javascript
module.exports = {
  theme: 'default',
  features: {
    analytics: true,
    darkMode: true
  }
}
```
