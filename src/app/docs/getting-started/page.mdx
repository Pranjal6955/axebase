# Getting Started with Axe.ai

This guide will help you get started with Axe.ai quickly and easily.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18.x or higher
- npm or yarn package manager
- Basic knowledge of React and Next.js

## Installation

### Using npm

```bash
npm install axebase
```

### Using yarn

```bash
yarn add axebase
```

### Using pnpm

```bash
pnpm add axebase
```

## Your First Project

Let's create your first Axe.ai project:

1. **Initialize your project**

```bash
npx create-axe-app my-app
cd my-app
```

2. **Start the development server**

```bash
npm run dev
```

3. **Open your browser**

Navigate to `http://localhost:3000` to see your app running!

## Project Structure

Your Axe.ai project will have the following structure:

```
my-app/
├── src/
│   ├── app/
│   │   ├── layout.tsx
│   │   └── page.tsx
│   └── components/
├── public/
├── package.json
└── next.config.js
```

## Configuration

Create a configuration file `axe.config.js` in your project root:

```javascript
module.exports = {
  theme: 'default',
  features: {
    analytics: true,
    darkMode: true
  }
}
```
